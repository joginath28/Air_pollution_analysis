# Air Pollution Analysis

## Project Overview
This project analyzes the impact of meteorological factors on air pollution (PM2.5).

## Objectives
- Analyze relationships between weather and pollution
- Identify key factors affecting PM2.5

## Dataset
- Weather variables (temperature, humidity, wind, etc.)
- Air quality indicators (PM2.5, PM10, CO, NO₂, etc.)

## Data Cleaning
- Removed duplicates
- Handled missing and invalid values
- Converted date/time formats

## Analysis
- Distribution plots
- Correlation heatmap
- Time-series trends
- Country-wise comparison

## Modeling
- Linear Regression
- 
## 🔍 Key Insights
- Visibility is the strongest predictor of PM2.5
- Rainfall reduces pollution levels
- Wind helps disperse pollutants
- Pollution varies across time and regions

## Model Performance
- R² Score: 0.08 (Linear Regression)
- Indicates pollution depends on additional external factors
